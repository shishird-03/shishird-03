### 🚀 Currently Building — Beyond Vision

> A Dual-Sensory Autonomous Navigation **ADAS** safety framework — my B.Tech final-year major project.

- **Visual branch**: U-Net semantic segmentation trained on Cityscapes, KITTI & BDD100K (unified 13-class labels)
- **Audio branch**: CNN-LSTM audio classifier trained on UrbanSound8K, ESC-50 & FSD50K
- **Stack**: PyTorch, OpenCV, librosa, Streamlit, Docker
- Working across a 4-person team covering CV, audio, fusion/integration, and DevOps
